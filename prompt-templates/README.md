# PaleoClaw Prompt Templates

常用研究场景的 Prompt 模板集合。

---

## 📚 文献研究

### 文献搜索模板
```
请帮我搜索关于 [研究主题] 的文献。

要求：
1. 优先检索 2018 年后的论文
2. 包含 DOI 链接
3. 按引用次数排序
4. 提供中英文摘要
```

### 文献综述模板
```
请帮我撰写关于 [研究主题] 的文献综述。

结构要求：
1. 引言（研究背景和意义）
2. 主要发现（分类归纳）
3. 研究趋势
4. 研究空白
5. 参考文献（APA 格式）
```

---

## 🦕 古生物学研究

### 物种分类查询
```
请查询 [物种名称] 的完整分类信息。

包括：
1. 界门纲目科属种
2. 地质年代（早/晚、期）
3. 主要化石产地
4. 文献依据
```

### 化石产地分析
```
请分析 [化石产地/地层] 的古生物学意义。

包括：
1. 地质年代
2. 岩性特征
3. 主要化石类群
4. 研究意义
5. 代表性标本
```

### 形态测量分析
```
请帮我分析 [化石标本图片] 的形态特征。

要求：
1. 提取地标点
2. 导出 TPS 格式
3. 生成可视化图
4. 与已知类群对比
```

## 📊 数据管理

### 数据导出
```
请帮我整理 [研究数据]。

输出格式：
1. CSV 格式表格
2. Excel 格式（含公式）
3. JSON 格式
4. 图表可视化
```

### BibTeX 生成
```
请将以下文献转换为 BibTeX 格式：

[文献列表或 DOI]

格式要求：
1. 标准 BibTeX 格式
2. 包含 DOI 链接
3. 中文文献特殊处理
```
