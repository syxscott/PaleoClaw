# `paleoclaw setup`

Initialize `~/.paleoclaw/paleoclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
paleoclaw setup
paleoclaw setup --workspace ~/.paleoclaw/workspace
```

To run the wizard via setup:

```bash
paleoclaw setup --wizard
```
