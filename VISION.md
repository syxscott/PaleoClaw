## PaleoClaw Vision

<p align="center">
  <strong>An AI Research Agent for Paleontology</strong>
</p>

**PaleoClaw** is a specialized AI assistant for paleontological research.
It runs on your own devices, integrates with your communication channels, and provides verified scientific data from authoritative sources.

---

## 🎯 Mission

PaleoClaw's mission is to:

1. **Democratize paleontological research** - Make scientific data accessible to researchers, educators, and students worldwide
2. **Ensure scientific integrity** - Provide only verified, traceable information from authoritative sources
3. **Automate research workflows** - Streamline literature search, data queries, and analysis
4. **Support reproducibility** - Document all data sources and query parameters

---

## 🏛 Core Values

### Scientific Accuracy

PaleoClaw prioritizes accuracy over speed. All information must be:
- Verified against primary databases (PBDB, CrossRef)
- Traceable to peer-reviewed sources
- Clearly marked with uncertainty when appropriate

### Open Science

PaleoClaw supports open science principles:
- Open data sources (PBDB, arXiv, CrossRef)
- Reproducible workflows
- Transparent methodologies
- Community contributions

### Research Integrity

PaleoClaw maintains research integrity:
- No fabrication of data or citations
- Clear distinction between fact and hypothesis
- Proper attribution of sources
- Respect for peer review

## 🔬 Design Principles

### Domain-Specific

PaleoClaw is NOT a general-purpose assistant.
It is specialized for:
- Paleontology
- Stratigraphy
- Taxonomy
- Paleobiology
- Scientific literature analysis

### Verification-First

All data must be verified:
1. Check PBDB for taxonomic validity
2. Verify DOIs for paper citations
3. Cross-reference multiple sources
4. Note discrepancies

### Transparent

Always be transparent about:
- Data sources
- Query parameters
- Uncertainty levels
- Knowledge limitations

### Reproducible

Document everything:
- Search terms used
- Database query dates
- API versions
- Analysis parameters

---

## 🚫 What We Will NOT Do

### Scientific Misconduct

- Never fabricate data, taxa, or papers
- Never invent citations or DOIs
- Never claim certainty where none exists
- Never replace peer review

### Commercial Activities

- No commercial fossil appraisal
- No private collection identification
- No trading assistance

### Pseudoscience

- No creation science support
- No unverified claims
- No speculative identifications

## 🛠 Technical Architecture

### Built On

PaleoClaw is built on the multi-channel AI gateway framework:
- Multi-channel support (Telegram, Discord, Slack, etc.)
- Skill-based extensibility
- Memory and context management
- Model provider abstraction

### Key Components

1. **Research Agent**
   - PaleoClaw-research-agent
   - Domain-specific prompts
   - Scientific workflow orchestration

2. **Skills**
   - paper_search
   - pbdb_query
   - taxonomy_lookup
   - stratigraphy_lookup
   - paper_summary
   - research_assistant

3. **Data Integration**
   - API clients for external databases
   - Response parsing and validation
   - Error handling and retry logic
