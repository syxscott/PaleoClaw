# Research Assistant Skill

Comprehensive paleontology research workflow automation.

## When to Use

✅ **USE this skill when:**

- "Write a literature review on [topic]"
- "Research summary for [taxon/group]"
- "Background research for [proposal/thesis]"
- "Comprehensive analysis of [formation/period]"
- Grant application background sections
- Thesis/dissertation literature chapters

## Workflow

The research assistant automatically executes this sequence:

```
User Query
    ↓
1. paper_search → Find relevant papers
    ↓
2. paper_summary → Analyze key papers
    ↓
3. taxonomy_lookup → Verify classifications
    ↓
4. pbdb_query → Query fossil occurrences
    ↓
5. stratigraphy_lookup → Get geological context
    ↓
6. Synthesize → Generate comprehensive report
```

## Scientific Integrity Rules

⚠️ **CRITICAL**:

- NEVER fabricate papers, taxa, or data
- If data unavailable: "No verified data found for [query]"
- Clearly mark uncertain or disputed information
- Distinguish hypothesis from established fact
- Cite all sources with DOI when available
- Note limitations of AI-generated summaries
- Recommend verification of critical claims

## Notes

- Reports are starting points, not final products
- Always consult primary literature
- Data reflects database state at time of query
- Some papers may require institutional access
