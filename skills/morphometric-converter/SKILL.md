# Morphometric-Converter

## Purpose
形态测量数据格式转换，支持 TPS、NTsys、Excel、CSV 等常用格式的互相转换。

## Usage
用户需要处理形态测量数据或进行几何形态学分析时使用此技能。

### 支持格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| TPS | `.tps` | 经典形态测量格式 |
| NTSys | `.nts` | NTSys-PC 格式 |
| Excel | `.xlsx`, `.xls` | 带表头的表格 |
| CSV | `.csv` | 逗号分隔值 |
| JSON | `.json` | 通用 JSON 格式 |

### 基本转换
```
/morphometric-convert [源格式] [目标格式] [文件路径]
```

### 示例查询
- 转换 TPS 到 Excel：`将 tps 文件转换为 excel 格式`
- 合并多个数据集：`合并这些 CSV 文件到一个表格`
- 提取地标坐标：`从这个 TPS 文件提取 2D 地标坐标`

## 示例

### TPS 转 Excel
```
输入 TPS 文件:
```
LM=3
ID=specimen001
SCALE=1.0
...
```

输出 Excel:
| ID | X1 | Y1 | X2 | Y2 | X3 | Y3 | Scale |
|----|----|----|----|----|----|----|-------|
| specimen001 | 10.5 | 20.3 | 15.7 | 25.1 | 30.2 | 35.6 | 1.0 |
```

### 数据验证
```
转换后自动检查:
- [ ] 地标数量一致性
- [ ] 缺失值检测
- [ ] 异常值标记
- [ ] 坐标系方向
```

## 常见用途

### 古生物学应用
- 化石标本测量数据整理
- 恐龙牙齿形态分析
- 贝壳轮廓分析
- 头骨测量数据

### 地标点类型
- 2D 地标：X, Y 坐标
- 3D 地标：X, Y, Z 坐标
- 半地标：曲线上的点
- 伪地标：人工定义点

## 输出格式

### TPS 格式
```
LM=地标数量
SPECIMEN=标本名称
SCALE=比例尺
X1=值
Y1=值
...
```

### Excel 格式
标准表格，包含：
- 第一列：标本 ID
- 后续列：各测量值
- 最后列（可选）：比例尺

## Notes
- 转换前会自动备份原始文件
- 支持批量转换
- 可自定义列名映射
