# Daily Log Generator Skill

Generate comprehensive daily activity logs in Markdown format.

## When to Use

✅ **USE this skill when:**

- "Generate my daily log"
- "What did I do today?"
- "Create my work report"
- "Show me my activity summary"
- "Generate daily productivity report"
- "Remember what I worked on today"
- "Where did I save that file?"

## When NOT to Use

❌ **DON'T use this skill when:**

- No activity data available
- Logging was disabled
- Data has been cleared

## Data Sources

1. **Activity Logger** - Application usage, file access
2. **Screen Monitor** - Screenshots, visual activity
3. **File System** - Modified files, new files
4. **Browser History** - Websites visited (with permission)
5. **Terminal History** - Commands executed
6. **Git Activity** - Commits, branches

## Commands

### Generate Daily Log

```bash
# Generate today's log
paleoclaw agent --message "Generate my daily log for today"

# Generate for specific date
paleoclaw agent --message "Generate my daily log for 2026-03-08"

# Generate with screenshots
paleoclaw agent --message "Generate my daily log with screenshots"
```

### Generate Weekly Report

```bash
# Weekly summary
paleoclaw agent --message "Generate my weekly activity report"
```

### Export Log

```bash
# Export to file
paleoclaw agent --message "Export my daily log to ~/Documents/daily-log.md"
```

## Configuration

### Log Generation Settings

```json
{
  "daily_log_generator": {
    "enabled": true,
    "auto_generate": true,
    "auto_generate_time": "18:00",
    "output_format": "markdown",
    "output_path": "~/paleoclaw-logs/daily-logs/",
    "include_screenshots": true,
    "include_file_changes": true,
    "include_websites": true,
    "include_commands": true,
    "include_productivity_score": true,
    "retention_days": 90
  }
}
```

### Export Settings

```json
{
  "export": {
    "formats": ["markdown", "json", "pdf"],
    "default_format": "markdown",
    "include_metadata": true,
    "compress_screenshots": true
  }
}
```

## Productivity Score Calculation

```
Score = (Focus Time / Total Time) × 100

Factors:
- Application switching frequency (-)
- Social media time (-)
- Deep work sessions (+)
- Task completion (+)
- Break regularity (+)
```
