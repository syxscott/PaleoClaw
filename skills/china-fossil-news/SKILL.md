# China-Fossil-News

## 目的
聚合中国化石新闻和最新发现，来自国家岩矿化石网站等权威来源。

## 使用方法
用户询问中国化石相关新闻或最新发现时使用此技能。

### 基本查询
```
/china-fossil-news [关键词]
```

### 示例查询
- 查询最新发现：`最新中国恐龙化石发现`
- 查询特定类群：`中国鸟类化石最新研究`
- 查询产地：`云南化石产地最新发现`
- 查询新闻：`热河生物群最新研究进展`

### 输出格式
```json
{
  "title": "新闻标题",
  "source": "来源网站",
  "date": "发布日期",
  "summary": "新闻摘要",
  "url": "原文链接",
  "tags": ["标签1", "标签2"]
}
```

## 数据来源

### 主要来源
- **国家岩矿化石网** (http://nkocs.com) - 中国地质博物馆主办
- **中国古生物学会** - 学术动态
- **各省地质博物馆** - 地方发现报道

### 新闻类型
| 类型 | 说明 |
|------|------|
| 重大发现 | 新物种、新产地报道 |
| 研究进展 | 最新研究成果发布 |
| 学术会议 | 学术活动报道 |
| 科普教育 | 科普活动新闻 |

## 功能特点

### 1. 多语言支持
- 中文标题和摘要
- 英文关键词搜索
- 地学术语双语对照

### 2. 分类筛选
- 按化石类群筛选（恐龙、鸟类、哺乳动物等）
- 按时代筛选（寒武纪、奥陶纪、石炭纪等）
- 按产地筛选（辽宁、云南、四川等）

### 3. 摘要生成
- 自动生成中文摘要
- 提取关键信息
- 标注重要程度

## 注意事项
- 新闻实时性：新闻可能在数小时至数天内更新
- 来源可靠性：所有新闻均来自权威机构
- 内容准确性：摘要基于原文自动生成，可能存在误差
